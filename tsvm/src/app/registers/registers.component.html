Instruction Set (x86)

Program Segements
  • .code: Location of code in program
  • .data: Location of data in program
Registers
  • r0 - r9: General Purpose Registers
  • ix: Interrupt Register
  • ip: Instruction Pointer
  • sp: Stack pointer register
  • flag: Flag register
Data
  • mov: Move data between registers (ex: mov r01 r1)
  • push: Push data onto stack (ex: push r0)
  • pop: Pop data from stack into a register (Ex: pop r0)
  • pusha: Push all registers (ex: pusha)
  • popa: Pop all registers (ex: popa)
Math
  • add: Add two registers, or register and value. (ex: add r0 r1)
  • sub: Subtract two registers, or register and value
  • div: Divide two registers, or register and value
  • mul: Multiply two registers, or register and value
  • inc: Increase register by 1
  • dec: Decrement register by 1
Bitwise
  • and: Logical AND two registers, or register and value
  • or: Logical OR two registers, or register and value
  • not: Logical NOT two registers, or register and value
  • xor: Logical XOR two registers, or register and value
  • neg: Negative value in register
Comparison
  • cmp: Compare two registers
Control Flow
  • jmp: Jump to label
  • je: Jump if last comparison equal x == y
  • jl: Jump if last comparison less than x < y
  • jg: Jump if last comparison greater than x > y
  • jne: Jump if last comparison not equal x != y
  • jle: Jump if last comparison less than equal x <= y
  • jge: Jump if last comparison greater than equal x >= y
Subroutine
  • <label>: A unique destination in memory
  • call: Pushes registers to stack. Jump to label
  • ret: Pop registers from stack. Return to instruction after to call
Interrupts
  • int 1: The screen
